# Font Download Instructions

The website uses locally hosted fonts for privacy. Please download the following fonts manually:

## Required Fonts

### Inter (from Google Fonts)
Download from: https://fonts.google.com/specimen/Inter

Select weights: 300, 400, 500, 600, 700, 800

### JetBrains Mono (from Google Fonts)
Download from: https://fonts.google.com/specimen/JetBrains+Mono

Select weights: 400, 500

## Alternative: Google Webfonts Helper

Use https://gwfh.mranftl.com/fonts to download fonts in woff2 format:

1. Search for "Inter"
2. Select charsets: latin
3. Select styles: 300, 400, 500, 600, 700, 800
4. Download and extract to this folder

5. Search for "JetBrains Mono"
6. Select charsets: latin
7. Select styles: 400, 500
8. Download and extract to this folder

## Expected Files

After download, this folder should contain:
- inter-v13-latin-300.woff2
- inter-v13-latin-regular.woff2
- inter-v13-latin-500.woff2
- inter-v13-latin-600.woff2
- inter-v13-latin-700.woff2
- inter-v13-latin-800.woff2
- jetbrains-mono-v18-latin-regular.woff2
- jetbrains-mono-v18-latin-500.woff2

Note: Filenames may vary slightly depending on download source. Update css/fonts.css if needed.
